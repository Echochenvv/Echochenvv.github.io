工作几年，慢慢攒出了一套自己顺手的工具链。不一定是最优解，但适合我的节奏。

## 代码编辑

VS Code 没什么好说的，插件生态太强了。几个必装的：GitLens、Error Lens、Pretty TypeScript Errors。

## 终端

用的 Warp，主要是颜值高，AI 补全也还行。之前用过 Alacritty + zsh，配置烦了就换回来了。

## 笔记

Obsidian，双链笔记的思路一旦习惯就回不去了。目前里面存了大概 500+ 条笔记，按项目和主题分了十几个文件夹。

## 设计

Figma 做原型，Excalidraw 画草图。有时候也会直接在 CodePen 里写 CSS 玩。

## 一个原则

工具够用就行，不要陷入「寻找完美工具」的陷阱。把时间花在产出上，不是配置上。
